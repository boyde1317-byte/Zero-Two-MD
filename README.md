## 🚀 Deployment

### Step 1 — Fork the Repository

[![Fork Repo](https://img.shields.io/badge/FORK%20REPOSITORY-brightgreen?style=for-the-badge&logo=github)](https://github.com/boyde1317-byte/Zero-Two-MD/fork)

### Step 2 — Get Your Session ID

1. Open WhatsApp on your phone
2. Go to **Settings → Linked Devices**
3. Tap **Pair Device** then run the bot and enter the pairing code shown in console

Or use the pairing command once connected:
```
.pair <your-number>
```

### Step 3 — Deploy

#### Railway
[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app)

#### Heroku
[![Deploy to Heroku](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com)

#### VPS / Local
```bash
git clone https://github.com/boyde1317-byte/Zero-Two-MD
cd Zero-Two-MD
npm install
npm start
```
